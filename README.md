## YouTube Plus 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `lyricLang` | `off` | 歌词翻译目标语言，ISO 639-1 代码，`off` 关闭 |
| `captionLang` | `off` | 字幕自动翻译目标语言，`off` 关闭 |
| `blockUpload` | `true` | 屏蔽上传类内容 |
| `blockImmersive` | `true` | 屏蔽沉浸式音乐内容 |
| `blockShorts` | `true` | 屏蔽 YouTube Shorts |
| `debug` | `false` | 开启调试日志 |

## YouTube Subtitles Translate 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `Type` | `Official` | 字幕类型，`Official` 或 `Translate` |
| `AutoCC` | `true` | 自动显示翻译字幕 |
| `ShowOnly` | `false` | 仅显示翻译字幕 |
| `Position` | `Forward` | 原始字幕位置，`Forward` 或 `Reverse` |
